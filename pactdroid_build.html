Flashing Nexus One with Cyanogenmod

  • Place Cyanogenmod ROM on root of SD card
  • Unlock bootloader: fastboot oem unlock
  • Boot into bootloader. Hold down trackball while powering on.
  • Flash AnonRa recovery image: fastboot recovery ~/path_to/recovery-RA-passion-v2.2.1-CM.img
  • Boot into AnonRa recovery. (Note: this is removed if stock ROM is installed)
  • Flash update-cm-7.1.0-N1-signed.zip from SD-card

Extract proprietary drivers from Nexus One

  • Flash N1 with Cyanogenmod7
  • Checkout AOSP
  • . build/envsetup.sh
  • cd device/htc/passion && ./extract-files.sh

Build custom AOSP for Nexus One

  • init repo
  • . build/envsetup.sh
  • lunch full_passion-eng
  • make update-api
  • make -j4
  • adb reboot fastboot
  • fastboot -w flashall
  • make sdk
